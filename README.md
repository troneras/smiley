# README

This project is a fun experiment to create images from ASCII characters. It can be served using the Uvicorn web server by running the command:

```shell
    uvicorn app:app --host 0.0.0.0 --port 8000
```
Enjoy creating ASCII art images!

 
You can also try it on GPTs if you have a premium account: https://chat.openai.com/g/g-gOyBljf6l-ascii-art-creator
